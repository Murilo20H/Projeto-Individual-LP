A programação surgiu com o desenvolvimento dos primeiros computadores, onde a programação era feita diretamente em linguagem de máquina.
Então em 1957 surge a 1º linguagem de programação, o FORTRAN.

FORTRAN

Desenvolvida pela IBM, foi a primeira linguagem de programação de alto nível, facilitando a escrita de códigos complexos para cálculos científicos e técnicos.

COBOL

Criada para aplicações comerciais, permitiu uma linguagem mais próxima do inglês, simplificando o desenvolvimento de sistemas empresariais.

LINGUAGEM C

Foi amplamente adotada devido à sua capacidade de escrever sistemas operacionais, compiladores e aplicativos de sistemas de forma eficaz.

ORIENTADA A OBJETOS

Na década de 1980 surgiram linguagens que introduziram conceitos de programação orientada a objetos, permitindo uma modelagem mais eficaz de sistemas complexos.

WEB E SCRIPT

O surgimento da internet impulsionou linguagens como JavaScript e PHP. Essas linguagens de script permitiram o desenvolvimento de páginas web dinâmicas e interativas.

LINGUAGENS MODERNAS

Linguagens como Python e Ruby ganharam popularidade devido à sua simplicidade e versatilidade. Além disso, paradigmas de programação têm sido explorados e implementados em linguagens modernas.
