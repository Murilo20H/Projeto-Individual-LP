Linguagens de Programação Populares

Python: usada em inteligência artificial, desenvolvimento web e análise de dados.
Java: usada em desenvolvimento de aplicativos Android, sistemas corporativos e aplicações de servidor.
JavaScript: usada em desenvolvimento web e aplicativos da web.
C++: usada em desenvolvimento de jogos, sistemas operacionais e software de alto desempenho.
SQL: usada em bancos de dados e consultas de dados.

Linguagens de Alto e Baixo Nível

Linguagens de alto nível: como Python e Java, são mais fáceis de entender e usar, pois se aproximam da linguagem humana. Elas simplificam a programação, mas podem ter menos desempenho.

Linguagens de baixo nível: como C e Assembly, são mais próximas da linguagem de máquina e oferecem mais controle sobre o hardware do computador, mas exigem um maior conhecimento técnico.

Paradigmas de Programação

Programação Imperativa: Centrada em "como" uma tarefa deve ser realizada, dividindo-a em uma sequência de instruções.

Programação Orientada a Objetos: Organiza o código em objetos que contêm dados e comportamentos relacionados.

Programação Funcional: Trata a computação como uma avaliação de funções matemáticas e evita estados mutáveis e efeitos colaterais.
