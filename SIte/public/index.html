Onde tudo começou?

O ábaco é um dos primeiros aparelhos de contas que se conhece, servia para fazer contas matemáticas simples, como somar, subtrair, multiplicar e dividir, usando contas móveis em hastes fixas.
O tear é uma ferramenta antiga de tecelagem. Ele usa fios para criar tecidos, movendo uma lançadeira entre eles para formar padrões. Apesar de não ser eletrônico, foi uma das primeiras máquinas a automatizar a tecelagem manual, marcando um avanço na indústria têxtil.
A máquina diferencial foi uma invenção do século XIX, projetada para cálculos automáticos, utilizava engrenagens e mecanismos. Apesar de nunca ter sido completamente construída, é vista como precursora dos computadores modernos.
